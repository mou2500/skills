# Phase 4: EXECUTE - Do The Work

## Overview

**Goal:** Produce quality output by completing planned tasks.

**When to Use:** After PLAN phase is complete.

**Success Criteria:** All tasks completed, ready for verification.

## The Process

### Step 1: Work One Task at a Time

**The Rule:** Complete current task before starting next.

**Why?**
- Maintains focus
- Prevents half-finished work
- Makes progress visible
- Reduces cognitive load

### Step 2: Show Your Work

For significant tasks, explain what you're doing:

```
**Starting Task 3: Draft context section**

I'm writing a brief, non-technical explanation of the integration issue...

[Show output]

**Task 3 Complete ✓**
Verified: Technical explanation is under 20 words
```

### Step 3: Mark Progress

Track completion explicitly:

```
**Progress: 3/5 tasks complete**
✓ Task 1: Draft opening
✓ Task 2: Draft context  
✓ Task 3: Draft response
→ Task 4: Draft ask and closing (in progress)
  Task 5: Full review
```

### Step 4: Flag Issues Immediately

If something unexpected happens:

```
**Issue Encountered in Task 4:**

The data export is missing March records. This affects our analysis.

**Options:**
A) Proceed with partial data (note limitation)
B) Investigate missing data source
C) Adjust analysis scope

Which would you prefer?
```

### Step 5: Adjust Plan When Needed

If plan needs revision:

```
**Plan Adjustment:**

Original: Task 5 was "Write conclusion"
New: Split into:
- Task 5a: Summarize findings
- Task 5b: Write recommendations
- Task 5c: Draft next steps

Reason: Conclusion is more complex than anticipated.

Proceeding with adjusted plan.
```

## Execution Patterns

### Pattern 1: Silent Execution (Simple Tasks)

For straightforward, low-risk tasks:

```
[Complete tasks 1-3 without showing each step]

**Tasks 1-3 Complete ✓**
- Gathered source materials
- Created outline
- Drafted introduction

Moving to Task 4...
```

### Pattern 2: Visible Execution (Complex Tasks)

For important or risky tasks:

```
**Task 4: Analyze pricing impact**

Grouping deals by price tier...
- High tier ($50k+): 12 deals, 75% win rate
- Mid tier ($20-50k): 28 deals, 60% win rate  
- Low tier (<$20k): 45 deals, 40% win rate

Comparing to Q1...
[Show analysis]

**Finding:** Win rate dropped most in low tier.

**Task 4 Complete ✓**
```

### Pattern 3: Batch Execution (Many Small Tasks)

For numerous similar tasks:

```
**Completing Tasks 5-10: Research phase**

[Work through all 6 tasks]

**Tasks 5-10 Complete ✓**
Summary of findings:
- 8 relevant sources identified
- 3 key themes emerged
- 2 contradictory studies noted

Moving to synthesis phase...
```

## Dealing with Blockers

### Missing Information

```
**Blocker in Task 6:**

I need [specific information] to proceed.

**Options:**
A) You provide the information
B) I make reasonable assumption (state it)
C) Skip this task, continue with others

What would you prefer?
```

### Technical Issues

```
**Issue in Task 4:**

The file format is different than expected.

**Solution:** Adapting approach to handle new format.

**Impact:** Task will take 5 extra minutes.

Proceeding with adjustment.
```

### Scope Creep

```
**Scope Question:**

While working on Task 7, I noticed [new requirement].

This wasn't in the original plan.

**Options:**
A) Add to current scope (adds 3 tasks)
B) Note for future iteration
C) Discuss and decide

What's your preference?
```

## Quality Checks During Execution

### Self-Review Checklist

Before marking task complete:

- [ ] Meets task definition
- [ ] Satisfies verification criteria
- [ ] Produces expected output
- [ ] No obvious errors
- [ ] Aligns with design intent

### Incremental Validation

For important tasks, validate before continuing:

```
**Task 4 Draft Complete**

[Show output]

Does this look right before I continue to Task 5?
```

## Anti-Patterns

### ❌ Jumping Ahead

```
"I see where this is going, let me skip to Task 8..."
```

**Problem:** May miss important steps, assumptions may be wrong

### ✅ Following the Plan

```
"Completing Task 3 as planned, then moving to Task 4..."
```

### ❌ Silent Execution of Complex Work

```
[30 minutes later]
"Done! Here's the complete analysis."
```

**Problem:** No opportunity for course correction

### ✅ Showing Progress

```
"Completed research phase. Here's what I found: [summary]

Moving to analysis phase next. This will take about 15 minutes."
```
